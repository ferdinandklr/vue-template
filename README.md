# Vue template

A basic vue template, including backend and REST API.
The objective is to implement on complete solution to build a website, including *(but not limited to)* :
 - frontend
 - backend
 - REST API
 - SQL - NOSQL
 - ...

## try for yourself

If you want to test this project, simply clone it, and run  ``npm install`` to install all dependencies, and then ``npm run dev-backend`` to start the server.
